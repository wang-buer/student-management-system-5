# 学生管理系统5

## 1、项目介绍

学生管理系统实现了管理员对学生和班级的增删改查，功能简单，适合新手学习


## 2、项目技术

后端框架： Servlet、mvc模式

前端技术：jsp、css、JavaScript

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql（5.x和8.x版本都可）
- 硬件环境：Windows 或者 Mac OS
